## General info
Simple command-line editor. Takes a directory as an argument and opens the files in that folder. Writing to files will be added in a later version, for now only reads files.

## Commands

| Commands     ||
|--------------|-----------|
| Exit program | Ctrl-Q    |
| Switch file  | Ctrl-W    |
